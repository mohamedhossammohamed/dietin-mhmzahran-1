# Dietin Web - Smart Diet & Workout Companion

Dietin is a comprehensive fitness application designed to help users track their nutrition, workouts, and health goals. Built with modern web technologies and wrapped for mobile using Capacitor, it offers a seamless experience across devices.

*   **Workout Plans**: Access and follow workout routines.

*   **AI Integration**: Features AI-powered suggestions (Gemini/OpenAI) for fit tips and meal planning.
*   **Cross-Platform**: Available on Web and Android (via Capacitor).
